Captionify

Caption images without jQuery

Download .zip View on GitHub

Basic usage

  1. Download Captionify from GitHub or install with Bower - bower install captionify.
  2. Add Captionify JS and CSS files to your document
  3. Initiate Captionify like so:
    <script>
    document.addEventListener("DOMContentLoaded", function(event) {
        var imageCaptions = new Captionify({
            containerSelector: ".pageContent"
        });
    });
</script>

This example will caption all images contained within element(s) with a .pageContent class. For more detailed documentation see the Github repo.
